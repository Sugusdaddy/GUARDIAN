# Changelog

## [2.2.0] - 2026-02-04

### Added
- **Global Statistics Page** - Comprehensive metrics dashboard
- **FAQ Page** - Frequently asked questions
- **Report Threats** - Community threat reporting system
- **Address Checker** - Check if addresses are blacklisted
- **Token Comparison** - Side-by-side security analysis
- **System Status Page** - Real-time agent monitoring
- **SVG Badges** - Embeddable security badges

- Real-time stat counters with animations
- Recent activity timeline
- Better error handling in API

### Added

- **Protection Proof** - NFT badge concept for secure wallets
- **Alerts System** - Watchlist with notifications
- **Token Leaderboard** - Ranked by security score

- Better risk analysis algorithm

  - Whale Tracker: Large holder monitoring

  - Oracle: ML risk prediction
  - + 8 more specialized agents

### Added
- **Landing Page** - Professional marketing site
- **API Documentation** - Full endpoint reference
- **Embeddable Widget** - For third-party sites
- **DexScreener Integration** - Real token data
