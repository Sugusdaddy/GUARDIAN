Integration Guide

1. For DEXs and Trading Platforms

Add pre-trade security checks to protect your users from honeypots and scams.

// Before executing a swap
const check = await guardian.honeypotCheck(tokenMint);
if (check.is_honeypot) {
  throw new Error('Token blocked: ' + check.reason);
}
// Proceed with swap...

2. For Other AI Agents

Subscribe to our threat feed to keep your users safe.

// Get active threats
const threats = await guardian.getThreats({ status: 'active' });

// Check if address is blacklisted
const isBad = await guardian.isBlacklisted(address);

3. For Wallets

Integrate emergency evacuation as a panic button feature.

// Emergency evacuation
await guardian.evacuate({
  source: compromisedWallet,
  destination: safeWallet,
  signTransaction: wallet.signTransaction
});
